******************************************
  zipaddr Japanese edition for Wordpress
******************************************

zipaddr日本語版(ジップアダー)は以下の環境で動作します。

Wordpress 3.3 以上 
PHP 5.x
MySQL 未使用 
文字コード UTF-8
また、http://zipaddr.com/とアプリケーションDB連携を行います。


【インストール】
解凍して得られたフォルダ「zipaddr-jp-tada」をWordpressのpluginsフォルダに設置してください。
アップデートする場合は「zipaddr-jp-tada」を上書きします。
一旦zipaddr_jp_tadaを停止してからアップデートし、有効化を行ってください。
単にデータを変換するだけですので、特に問題はないと思います。
詳細は、http://zipaddr.com/wordpress/ を参照して下さい。

zipaddr_jp_tadaを使用中に起こった如何なる損害も自己責任となります。


【開発環境】
OS 未定
PHP 5.2
MySQL 5.1 
Apache 未定
